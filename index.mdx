ZLS is a non-official Language Server Protocol (LSP) implementation for Zig, written in Zig itself. It connects to any LSP-compatible editor — VS Code, Neovim, Helix, Zed, and many more — to provide rich IDE features like autocompletion, goto definition, hover documentation, and real-time diagnostics powered by Zig's own compiler.

## Features at a glance

ZLS implements the core LSP features that make Zig development faster and less error-prone:

<CardGroup cols={3}>
  <Card title="Completions" icon="wand-magic-sparkles" href="/features/completions">
    Smart autocompletion with snippets, argument placeholders, and function signatures.
  </Card>
  <Card title="Diagnostics" icon="triangle-exclamation" href="/features/diagnostics">
    Inline errors and warnings via `zig ast-check` and optional build-on-save.
  </Card>
  <Card title="Navigation" icon="arrow-right-arrow-left" href="/features/navigation">

  <Card title="Hover" icon="circle-info" href="/features/navigation">

## Get up and running

<Steps>
  <Step title="Install ZLS">
    Download a prebuilt binary matching your Zig version from the [ZLS releases page](https://github.com/zigtools/zls/releases), or build from source with `zig build -Doptimize=ReleaseSafe`. See the [installation guide](/installation) for all options.
  </Step>
  <Step title="Connect to your editor">
    Follow the [editor setup guide](/editor-setup) for your editor. Most editors need only a path to the `zls` binary and the LSP client enabled.
  </Step>
  <Step title="Configure (optional)">
    Create a `zls.json` in your config directory to tune completions, diagnostics, and inlay hints. See the [configuration guide](/guides/configuration) or the full [configuration reference](/reference/configuration).
  </Step>
  <Step title="Start coding">
    Open a Zig file and ZLS will start automatically. Trigger completions, hover over symbols, and navigate your project with full IDE support.
  </Step>
</Steps>

<Note>
  ZLS version must match your Zig version. If you upgrade Zig, upgrade ZLS at the same time. The `main` branch of ZLS tracks Zig `master`.
</Note>
